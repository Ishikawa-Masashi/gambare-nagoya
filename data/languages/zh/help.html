Kopanito是一款技能游戏。也就是说，你必须在游戏中不断地学习技能以求进步(就像是现实生活中的足球)。你需要不断练习你的精准度和反应力以成为一名更好的球员。运气在Kopanito游戏中并不管用。比较强的球员总是能不断赢得比赛，这里并没有数据或捷径能帮助你赢得比赛。你就是自己不断创造的新数据。

使用 {{ 'up'|action }}{{ 'down'|action }}{{ 'left'|action }}{{ 'right'|action }} 来移动你的球员 (下方有一个圆圈的那个)。

{{ 'pass'|action }} 地面传球
{{ 'togglePlayer'|action }} 更换球员
{{ 'shoot'|action }} 射门 (长按可使用精确射门模式)
{{ 'header'|action }} 空中射门(当你在球下方时长按)
{{ 'slide'|action }} 铲球
{{ 'lob'|action }} 高吊传球 (長按以做強力踢球)

传球

球员间有两种传球方式：地面传球与高吊传球。

地面传球

按下 {{ 'pass'|action }} 来做地面传球。使用 {{ 'up'|action }}{{ 'down'|action }}{{ 'left'|action }}{{ 'right'|action }} 来控制传球的方向。系统会自动选择接球的队员。

高吊传球

长按 {{ 'lob'|action }} 来做高吊传球。使用{{ 'up'|action }}{{ 'down'|action }}{{ 'left'|action }}{{ 'right'|action }} 来控制传球的方向。按键的时间越长传球的距离越远。

辅助传球 vs 手动传球

辅助传球指的是由电脑控制接球的球员(在可行的情况下)。这是一种电脑自动的传球方式。

辅助传球使你能沿贯你的动作。例如：将球传的更远或是在队友停接球前就先射门。你可以在游戏的练习模式中使用这项技巧。

相反的，手动传球能让你更精确的掌握踢球方向并且在球被拦下前选择接球球员。手动传球时也可使用悬空踢球。

在踢球时长按传球组合键 {{ 'passModifier'|action }} 以达成手动传球。

注意：辅助传球模式时使用头踢或倒勾球踢在大多数情况下是不可能的，你还是需要使用手动高吊传球来达成这个目标。

注意 2：仅使用辅助传球模式来尝试打败职业级或传奇AI不是个好主意，在此警告。

射门

使用 {{ 'shoot'|action }} 来射门。以下有几种射门的方法。

直接射门

快速按下 {{ 'shoot'|action }}，系统会自动帮你的球员依照最好的方向射门。但射门的力道并不会很强。这种射门是诱使对方守门员离开球门的最好选择。

點傳射门

这种射门方式的力道比直接射门强很多，但踢球的方向仍然是由系统决定。这种方式适合用在没人持球的时后。

精准射门

当你长按射门键，你将会进入精确射门的模式。你能使用{{ 'up'|action }}{{ 'down'|action }}{{ 'left'|action }}{{ 'right'|action }}键来调整踢球的方向并且控制球的转弯。按键的时间越长，射门的力道将会越大。

可以透过长按 {{ 'swerveUp'|action }} 或 {{ 'swerveDown'|action }} 来弯曲瞄准线以使球转弯。 这是用来误导对方守门员或是当你从狭窄角度射击时的好方法。

提示：强力的射门可能击倒其他球员，包括守门员。

铲球

按下 {{ 'slide'|action }} 以对你移动的方向铲球。这是一个非常强大的防守技巧，但你需要良好的精确度，尤其是当你在试图拦截移动中的对手。是的，针对对手而不是球是一个非常阴险而令人惊讶的策略。例如，尝试预测对手何时会传球，并在球到达之前击倒接球手。

铲球还可以使你"跑"的更快(不能带球)、穿越过守门员来获得进球，或取消精确射门。

悬空踢球

当球在空中时你可以藉由接近球并长按 {{ 'header'|action }} 来施行有力的头踢或倒勾球踢。踢球方向和直接射门一样为电脑自动控制。

记住你也可以在防守时使用这个动作，像是当有对手进入你的罚球区时。

超级动作

在屏幕的上部有两个技能条，在比赛期间逐渐填满。其中一个技能条是你球队的，另一条是对手的。当技能条填满时，将出现发光图标并且你会得到一个随机的超级技能。

按下 {{ 'powerup'|action }} 以启动超级动作。有时后根据不同的超级移动，你需要按住该按钮更长时间。

有些超级动作能用来抵消对手的超级动作。

吸球磁铁

使用它来把球吸给你正在控制的球员。适合制造惊喜，可以用于进攻和防守。

瞬移

长按 {{ 'powerup'|action }} 后十字准线会出现。放开按钮后将会把您当前控制的玩家传送到该地点。 你可以和球一起传送到对手的得分区或传送到持有球的对手旁。这也可以用来阻止对手的射门。注意：在瞄准期间，对手可以从你身上偷走球

超级射门

这是一个非常强大的踢球，之后你可以使用 {{ 'up'|action }}{{ 'down'|action }}{{ 'left'|action }}{{ 'right'|action }} 来控制球。但试着避免打到任何球员，因为这样会减慢球的速度。这项技巧甚至可以从你的半场进行攻击并且仍能得分。

旋风制造

旋风制造将会把球吹出你的球门外。使用这个招式以防守对方的射门。但此技能只能维持几秒钟。当你时间抓准时就能阻止超级射门。

冻结

"冻结"将会使周遭球员(包括你的队员!)固定不动一阵子。当球在"冻结"的范围内将会停止，这能帮助你阻止对手的进攻。

你能对你的队友使用铲球动作以帮助他解除冻结。

特斯拉直升机

特斯拉直升机总是跟随带球的球员，并允许他电击其半径内的其他球员。一共有4次电击机会，在最后一次使用后特斯拉直升机便会飞走。

注意：失去球意味着也失去了特斯拉直升机的特技！

守门员

守门员仅由AI控制。当他们是离球最近的队员时，他们会尝试抓住球或拦截球。你可以使用 {{ 'goalkeeperSlide'|action }} 让守门员来执行铲球，但请小心使用。当守门员做铲球时，他总是朝着球的方向前进。但如果球移动得很快，守门员大部分时间都会错过球。

当你的守门员拥有球时，你可以使用适当的按钮 ({{ 'pass'|action }} and {{ 'lob'|action }}) 来做一般传球或高吊传球。注意：你有时间的限制－3秒。如果你在给定的时间内没有传球，AI 就会控制你的守门员。

铲球对守门员是无效的。但守门员无法带球移动。

高级技巧

加速

长按 {{ 'sprint'|action }} 以加速。这只在玩家没有持球的情况下能使用。球员没有耐力的问题而且永远不会疲劳，所以加速的使用是无限制的。

向前推进

当你快速按下 {{ 'pass'|action }} 你将能把球向前推进。这个技能在当你的玩家前面没有球员时才有效。然后你可以透过冲刺赶上球。

更换球员

当你使用 {{ 'togglePlayer'|action }} 更换球员时，使用{{ 'up'|action }}{{ 'down'|action }}{{ 'left'|action }}{{ 'right'|action }} 向你所认定的球员指出球的方向是很重要的。如果你没按下箭头键，游戏将选择最接近球的球员。

蹭球射门

只需要简单按下 {{ 'lob'|action }} 你便可以从守门员上方传球。最好的时机为当对方守门员跑出射门范围的时后。

其他提示

带球的球员移动比较慢并且不能冲刺。 如果对手在你带球时撞到你，那么他将会从你身上把球偷走。当然，带有球的玩家会因位对手精确的铲球而被击倒，因此速度和准确度对于维持球权是很重要的。 如果你有带球的问题，那么你该考虑更多的传球，因为移动中的球通常比任何对手都来的快。
